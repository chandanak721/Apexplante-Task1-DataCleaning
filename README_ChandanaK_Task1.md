**Intern Name: Chandana K**\

**Domain: Data Analytics Internship**

## Objective

To understand the provided sales dataset, perform data quality
assessment, cleaning, transformation and create an analysis-ready
dataset for further exploratory data analysis.

## Dataset Overview

The dataset represents sample sales transactions containing information
about: - Orders and dates\
- Customers\
- Product categories\
- Sales revenue and profit\
- Quantity and region

## Steps Performed

### 1. Data Familiarization

-   Loaded raw dataset using Python Pandas\
-   Reviewed structure, columns and data types

### 2. Data Dictionary

-   Prepared data dictionary explaining each column, type and business
    relevance

### 3. Data Quality Assessment

-   Identified missing values\
-   Checked duplicate records\
-   Verified inconsistent formats\
-   Detected outliers in Profit column

### 4. Data Cleaning

-   Imputed missing sales values using mean\
-   Removed duplicate rows\
-   Standardized category text\
-   Converted Order_Date to datetime format

### 5. Feature Engineering

-   Created Year and Month columns from Order_Date\
-   Prepared metrics useful for future KPI analysis

### 6. Outlier Treatment

-   Applied IQR method on Profit column to remove extreme values

## Tools Used

-   Python\
-   Pandas\
-   Jupyter Notebook

## Deliverables

-   Raw dataset\
-   Data dictionary\
-   Cleaning script\
-   Cleaned analysis-ready dataset

## Outcome

The final dataset is structured, consistent and ready for Exploratory
Data Analysis and dashboard creation in Task‑2.
